# Computer Vision at SafetyCulture — Opportunity Solution Tree

An interactive Opportunity Solution Tree (OST) mapping the problem space, opportunities, and potential solutions for Computer Vision at SafetyCulture.

## View Online

[Open the Opportunity Solution Tree](https://krisurban-design.github.io/opportunity-solution-tree/CV_Opportunity_Solution_Tree.html)

> Note: The live link is only accessible to collaborators with repo access.

## About

This OST is a visual product discovery tool used to align the team around:
- The **desired outcome** we are working towards
- **Opportunities** (user needs, pain points, and desires)
- **Solutions** being explored to address those opportunities

## How to Use

Open the link above in any browser — no installation required.
